### 问题

- [x] Grue 是按照常见音译 **格鲁** 还是按照意译 **暗魇**？一般而言披风的名字都应该意译，例如武器大师、谜探等，它们通常与超能力有所关联 -- 采用意译保持一致性
- [x] 在 PHO 上的谜探（原告密者）缩写 TT 怎么翻译？-- 保留原文，本身就已经好记了，也和“探探”发音首字母一致
- [x] costume 应该翻译为 戏服 还是 披风服装？-- 戏服 看起来是个直译，但确实听起来不错，比披风服装听起来更有趣
- [ ] 在内容中模拟 PHO 论坛帖子要怎么处理？一般都有缩紧，使用 >？-- 暂时使用 GitHub Alerts 语法，后续如果需要也可以批量转换为 Custom Container 语法

  ```md
  ::: info 标题：虫

  欠你一个人情。想要报答。见面？

  发消息，

  TT。
  :::

  > [!NOTE] 标题：虫
  >
  > 欠你一个人情。想要报答。见面？
  >
  > 发消息，
  >
  > TT。
  ```
