主角是一个名叫泰勒・赫伯特的女高中生，是个失去母亲的黑发褐肤、性格阴郁的女孩。她在学校遭受霸凌，唯一支撑她活下去的希望是：不久前她经历了“触发事件”，获得了控制昆虫的超能力。如果她能顺利参加“监护者计划”，就能成为一个真正的超级英雄，为她那悲惨的人生增添一些意义...
